题解 [CQOI2014]数三角形 洛谷P3166

发布于 2020-07-16 | 分类于 欧拉反演 、 组合数学 、 题解 | 13分钟 | 2386字数

背景

提供一个相当美妙并且相当好写的 O(n)O(n)O(n) 做法。

思路

Part I

显然，总体思路应为：「三角形数量」等于「任选三个点的方案数」减去「三点共线的方案数」。

共有 (n+1)(m+1)(n+1)(m+1)(n+1)(m+1) 个点，故「任选三个点的方案数」为 C(n+1)(m+1)3\mathrm{C}_{(n+1)(m+1)}^3C(n+1)(m+1)3​。

「三点共线的方案数」等于「横着的」加「竖着的」加「斜着的」。

「横着的」指三点所在直线平行于 xxx 轴。共有 m+1m+1m+1 条横线，每条横线上有 n+1n+1n+1 个点，从这 n+1n+1n+1 个点中选取 333 个，方案数为 (m+1)⋅Cn+13(m+1)\cdot \mathrm{C}_{n+1}^3(m+1)⋅Cn+13​。

同理，「竖着的」指三点所在直线平行于 yyy 轴，方案数为 (n+1)⋅Cm+13(n+1)\cdot \mathrm{C}_{m+1}^3(n+1)⋅Cm+13​。

看来，我们现在最关心的是「斜着的」，斜着的直线按斜率正负分为两种，并且这两种的方案数是相等的。因此，我们只需要计算出斜率为正的方案数，再乘以 222 即可。

方便起见，不妨将核心计算内容——斜率为正的方案数记为 ansansans。

Part II

不妨先考虑最暴力的做法，无脑枚举所有不同的的三个点，判断两两斜率是否相等。

进一步思考我们发现，相对位置确定的三个点可以在网格中随意平移、反复横跳，导致白白浪费了算力。

那么可不可以先枚举这三个点的相对位置关系，再计算有这样相对位置关系的三元点对有多少个呢？

注意，一旦相距最远的两个点横坐标差 iii、纵坐标差 jjj 确定了，无论第三个点选取在这两点连线上的哪个整点，对应的三元点对数量都是 (n−i+1)(m−j+1)(n-i+1)(m-j+1)(n−i+1)(m−j+1)，这是因为左右平移有 n−i+1n-i+1n−i+1 种选择，上下平移有 m−j+1m-j+1m−j+1 种选择。

想到了这里，也就不难发现一个小结论（手玩几组数据就可以发现，证明见文末）：假设网格中 ABABAB 平行于底边 xxx 轴，长度为 iii ，BCBCBC 平行于侧边 yyy 轴，长度为 jjj ，那么 ACACAC 上的整点数量为 gcd⁡(i,j)+1\gcd(i,j)+1gcd(i,j)+1。

对于每组横坐标差 iii，纵坐标差 jjj 的点对，我们先选取两端点，再在他俩之间的 gcd⁡(i,j)−1\gcd(i,j)-1gcd(i,j)−1 个点中任选一个作为第三个点，方案数为 gcd⁡(i,j)−1\gcd(i,j)-1gcd(i,j)−1。综上：

ans=∑i=1n∑j=1m(n−i+1)(m−j+1)(gcd⁡(i,j)−1)ans=\sum_{i=1}^n\sum_{j=1}^m(n-i+1)(m-j+1)(\gcd(i,j)-1) ans=i=1∑n​j=1∑m​(n−i+1)(m−j+1)(gcd(i,j)−1)

直接计算这个式子复杂度 O(nm)=O(n2)O(nm)=O(n^2)O(nm)=O(n2)，考虑 gcd⁡\gcdgcd 的计算的话甚至是 O(n2log⁡n)O(n^2\log n)O(n2logn)。当然，对于本题的数据是绰绰有余了。

Part III

我们看到 [gcd⁡(i,j)=1][\gcd(i,j)=1][gcd(i,j)=1] 这样的项时，总是按照莫比乌斯反演套路将其化为 ∑d∣gcd⁡(i,j)μ(d)\sum_{d\mid\gcd(i,j)}\mu(d)∑d∣gcd(i,j)​μ(d)。

实际上，对于 gcd⁡(i,j)\gcd(i,j)gcd(i,j) 这样的项时，也是有套路的，那就是根据欧拉反演将其化为 ∑d∣gcd⁡(i,j)φ(d)\sum_{d\mid\gcd(i,j)}\varphi(d)∑d∣gcd(i,j)​φ(d)。

他的原理是欧拉函数的一个有趣性质：

nnn 的所有因子的欧拉函数和为 nnn ，即 ∑d∣nφ(d)=n\sum_{d\mid n}\varphi(d)=n∑d∣n​φ(d)=n。

证明及更详细的信息见 ∅\varnothing∅ 的博客。

也就是说：

ans=∑i=1n∑j=1m(n−i+1)(m−j+1)(∑d∣gcd⁡(i,j)φ(d)−1)ans=\sum_{i=1}^n\sum_{j=1}^{m}(n-i+1)(m-j+1)\left(\sum_{d\mid\gcd(i,j)}\varphi(d)-1\right) ans=i=1∑n​j=1∑m​(n−i+1)(m−j+1)⎝⎛​d∣gcd(i,j)∑​φ(d)−1⎠⎞​

考虑到 φ(1)=1\varphi(1)=1φ(1)=1，不妨先预先化简下，把 φ(1)\varphi(1)φ(1) 和 −1-1−1 抵消掉，免得以后麻烦：

ans=∑i=1n∑j=1m(n−i+1)(m−j+1)∑d∣gcd⁡(i,j)d≠1φ(d)ans=\sum_{i=1}^n\sum_{j=1}^{m}(n-i+1)(m-j+1)\sum_{d\mid\gcd(i,j)}^{d\ne 1}\varphi(d) ans=i=1∑n​j=1∑m​(n−i+1)(m−j+1)d∣gcd(i,j)∑d​=1​φ(d)

更复杂了？不妨先来看看 d∣gcd⁡(i,j)d|\gcd(i,j)d∣gcd(i,j) 的含义：

i,ji,ji,j 的最大公约数的每个因子。

那不正是 i,ji,ji,j 的每个公约数吗？

枚举 i,ji,ji,j 再枚举他们的所有公约数，等价于枚举每个约数 ddd 再枚举 ddd 的 iii 倍、jjj倍。

可以看出，ddd 的取值范围成了 [2,min⁡(n,m)][2,\min(n,m)][2,min(n,m)]，i,ji,ji,j 最小可取 111，最大分别取 ⌊nd⌋,⌊md⌋\left\lfloor\dfrac{n}{d}\right\rfloor,\left\lfloor\dfrac{m}{d}\right\rfloor⌊dn​⌋,⌊dm​⌋，即：

ans=∑d=2min⁡(n,m)∑i=1⌊n/d⌋∑j=1⌊m/d⌋(n−id+1)(m−jd+1)φ(d)ans=\sum_{d=2}^{\min(n,m)}\sum_{i=1}^{\lfloor{n/d}\rfloor}\sum_{j=1}^{\lfloor{m/d}\rfloor}(n-id+1)(m-jd+1)\varphi(d) ans=d=2∑min(n,m)​i=1∑⌊n/d⌋​j=1∑⌊m/d⌋​(n−id+1)(m−jd+1)φ(d)

=∑d=2min⁡(n,m)φ(d)∑i=1⌊n/d⌋(n−id+1)∑j=1⌊m/d⌋(m−jd+1)=\sum_{d=2}^{\min(n,m)}\varphi(d)\sum_{i=1}^{\lfloor{n/d}\rfloor}(n-id+1)\sum_{j=1}^{\lfloor{m/d}\rfloor}(m-jd+1) =d=2∑min(n,m)​φ(d)i=1∑⌊n/d⌋​(n−id+1)j=1∑⌊m/d⌋​(m−jd+1)

很明显，∑i=1⌊n/d⌋(n−id+1)\sum_{i=1}^{\lfloor{n/d}\rfloor}(n-id+1)∑i=1⌊n/d⌋​(n−id+1) 是等差数列求和，

首项为 (n−d+1)(n-d+1)(n−d+1)，末项为 (n−⌊nd⌋d+1)(n-\left\lfloor\dfrac{n}{d}\right\rfloor d+1)(n−⌊dn​⌋d+1) 即 (n mod d+1)(n\bmod d+1)(nmodd+1)，项数为 ⌊nd⌋\left\lfloor\dfrac{n}{d}\right\rfloor⌊dn​⌋，故：

∑i=1⌊n/d⌋(n−id+1)=12(n−d+1+n mod d+1)⌊nd⌋\sum_{i=1}^{\lfloor{n/d}\rfloor}(n-id+1)=\dfrac{1}{2}(n-d+1+n\bmod d+1)\left\lfloor\dfrac{n}{d}\right\rfloor i=1∑⌊n/d⌋​(n−id+1)=21​(n−d+1+nmodd+1)⌊dn​⌋

同理有：

∑j=1⌊m/d⌋(m−jd+1)=12(n−d+1+n mod d+1)⌊nd⌋\sum_{j=1}^{\lfloor{m/d}\rfloor}(m-jd+1)=\dfrac{1}{2}(n-d+1+n\bmod d+1)\left\lfloor\dfrac{n}{d}\right\rfloor j=1∑⌊m/d⌋​(m−jd+1)=21​(n−d+1+nmodd+1)⌊dn​⌋

最终得到：

ans=14∑d=2min⁡(n,m)φ(d)(n−d+n mod d+2)⌊nd⌋(m−d+m mod d+2)⌊md⌋ans=\dfrac{1}{4}\sum_{d=2}^{\min(n,m)}\varphi(d)(n-d+n\bmod d+2)\left\lfloor\dfrac{n}{d}\right\rfloor(m-d+m\bmod d+2)\left\lfloor\dfrac{m}{d}\right\rfloor ans=41​d=2∑min(n,m)​φ(d)(n−d+nmodd+2)⌊dn​⌋(m−d+mmodd+2)⌊dm​⌋

欧拉函数可以利用欧拉筛线性预处理出，枚举 ddd 复杂度 O(min⁡(n,m))=O(n)O(\min(n,m))=O(n)O(min(n,m))=O(n)。

代码

#include <cstdio>
#include <iostream> 
using namespace std;
const int maxn = 1005;
int n, m, p[maxn], phi[maxn], tot;
bool mark[maxn];
long long ans;
long long C(long long x) { return x * (x-1) * (x-2) / 6; }
void sieve(int n) {
	phi[1] = 1;
	for(int i = 2; i <= n; ++i) {
		if(!mark[i]) p[++tot] = i, phi[i] = i - 1;
		for(int j = 1; j <= tot and p[j]*i <= n; ++j) {
			mark[p[j]*i] = true;
			if(i % p[j]) phi[p[j]*i] = phi[i] * (p[j]-1);
			else { phi[p[j]*i] = phi[i] * p[j]; break; }
		}
	}
}
int main() {
	scanf("%d %d", &n, &m);
	if(n > m) swap(n, m);
	sieve(n);
	for(int d = 2, x, y; d <= n; ++d)//记得 ans 要乘以 2，这里直接除以 2 就行了不用除以 4
		ans += (long long)phi[d]*(n-d+n%d+2)*(n/d)*(m-d+m%d+2)*(m/d)/2;
	ans = C((n+1)*(m+1)) - (m+1)*C(n+1) - (n+1)*C(m+1) - ans;
	printf("%lld\n", ans);
}

P.S.

估计相当一部分小朋友在初中的数学竞赛中就接触过了这个结论，至少我是这样的。

这其实是属于那种「这也用证」的，以下口胡一个证明：

求证：假设网格中 ABABAB 平行于底边 xxx 轴，长度为 aaa ，BCBCBC 平行于侧边 yyy 轴，长度为 bbb ，那么 ACACAC 上的整点数量为 gcd⁡(a,b)+1\gcd(a,b)+1gcd(a,b)+1。

证明：不妨令 AAA 点为坐标原点，BBB 为 (a,b)(a,b)(a,b)。设 gcd⁡(a,b)=d\gcd(a,b)=dgcd(a,b)=d，且 a=pd,b=qd,a,b,d,p,d∈N+,p⊥qa=pd,b=qd,a,b,d,p,d\in\mathbb{N}_+,p\perp qa=pd,b=qd,a,b,d,p,d∈N+​,p⊥q。

则两点连线上的整点 (x,y)(x,y)(x,y) 必然为满足 x,y∈N+,x∈(0,a),y∈(0,b)x,y\in\mathbb{N}_+,x\in(0,a),y\in(0,b)x,y∈N+​,x∈(0,a),y∈(0,b) 且 yx=ba=qp\dfrac{y}{x}=\dfrac{b}{a}=\dfrac{q}{p}xy​=ab​=pq​，即 xq=ypxq=ypxq=yp，又 ∵p⊥q,∴p∣x,q∣y\because p\perp q,\therefore p\mid x,q\mid y∵p⊥q,∴p∣x,q∣y。

设 x=kp,y=k′q,k,k′∈N+,k,k′∈[1,d−1]x=kp,y=k'q,k,k'\in\mathbb{N}_+,k,k'\in[1,d-1]x=kp,y=k′q,k,k′∈N+​,k,k′∈[1,d−1]，又 ∵kpq=k′qp,∴k=k′\because kpq=k'qp,\therefore k=k'∵kpq=k′qp,∴k=k′，故 kkk 的数量即为整点数量，又 ∵k∈[1,d−1]\because k\in[1,d-1]∵k∈[1,d−1]，∴\therefore∴ 整点数为 gcd⁡(a,b)−1\gcd(a,b)-1gcd(a,b)−1。
